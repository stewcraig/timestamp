Timestamp miscroservice

Microservice that provides unix timestamp and natural languge date for a given date or timestamp.

Unix timestamp is the number of milliseconds since 1 January 1970.

Use

Enter date or unix time stamp in address bar of browser. See example inputs below.

Natural date:

https://timestampmicrosvc.herokuapp.com/1 January 2014

Unix date:

https://timestampmicrosvc.herokuapp.com/1388494800000

Output

Unix timestamp and natural lanaguage date will be output in the format below.

{"unix": 1388494800000, "natural":"1 January 2014"}

If format of input is not recognised then null will be returned for both unix timestamp and natural date.
